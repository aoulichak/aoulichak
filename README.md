### `/projects`

<table>
<tr>
<td width="50%" valign="top">

#### [Intelligent Handwriting Character Recognition](https://github.com/aoulichak/intelligent-handwriting-character-recognition)

An end-to-end **ICR system** for classifying handwritten alphabets, evolved through 3 versions — from a baseline **PyTorch CNN** to an advanced **Keras** model with a modernized GUI. Covers the full pipeline: data preprocessing, model training, evaluation, and real-time inference.

`Python` `PyTorch` `Keras` `TensorFlow` `CNN`

</td>
<td width="50%" valign="top">

#### [OCR University Project](https://github.com/aoulichak/OCR-University-Project)

A complete OCR + ICR suite. **V1** integrates Tesseract and Google Gemini AI for multilingual text extraction with intelligent correction. **V2** uses a custom **CNN (PyTorch)** achieving >90% accuracy on handwritten character recognition with a real-time drawing interface.

`Python` `PyTorch` `Tesseract` `Gemini API` `OpenCV`

</td>
</tr>
<tr>
<td width="50%" valign="top">

#### [Tesseract Image Processor](https://github.com/aoulichak/Tesseract-Image-Processor)

A hybrid OCR pipeline combining **Tesseract** for text extraction and **Google Gemini AI** for post-processing. Automatically corrects OCR errors, classifies document types (invoices, contracts, CVs...), and supports multilingual processing (FR/AR/EN) through a professional **PySide6** interface.

`Python` `Tesseract OCR` `Gemini API` `PySide6`

</td>
<td width="50%" valign="top">

#### [Handwritten A2Z Synthetic Pipeline](https://github.com/aoulichak/Handwritten-A2Z-Synthetic-Pipeline)

A large-scale data generation pipeline that produced **359,000 synthetic handwritten images** (28x28 grayscale) across 26 uppercase classes. Uses TRDG & Google Fonts with heavy augmentation (rotation, noise, blur, skew) to bridge the gap between digital fonts and real handwriting.

`Python` `TRDG` `Pillow` `NumPy` `Data Augmentation`

</td>
</tr>
<tr>
<td width="50%" valign="top">

#### [FactureProWeb](https://github.com/aoulichak/FactureProWeb)

A modern **invoice management web application** built for professionals and small businesses. Handles the full invoicing cycle — client & product management, professional PDF generation, advanced statistics and reporting — with a clean, responsive interface.

`Python` `Flask` `HTML` `CSS` `PDF Generation`

</td>
